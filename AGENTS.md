# Instrucciones para agentes

## Fuente de verdad inicial

Al empezar una tarea en este repositorio:

1. Lee `README.md`.
2. Lee `TODO.md`.

## Regla de mantenimiento del TODO

Siempre que hagas una tarea, revisa `TODO.md` y actualizalo si es necesario.

`TODO.md` y `AGENTS.md` forman parte del repo y deben quedar listos para retomar el trabajo en otra sesion.

Esto incluye, por ejemplo:

- marcar tareas hechas o ya empezadas
- ajustar la siguiente prioridad
- anadir tareas nuevas descubiertas durante el trabajo
- eliminar o reformular tareas que ya no tengan sentido
- dejar claro cual es el siguiente paso recomendado si el trabajo queda a medias

No dejes el `TODO.md` desactualizado si el trabajo realizado cambia el estado real del proyecto.

## Criterios editoriales

- Escribe en castellano.
- Usa texto claro, corto y util.
- Indica siempre autenticacion, gratuidad, oficialidad y notas importantes.
- No marques como `Verificada` una API dudosa.
- Prioriza APIs de Espana y Latinoamerica.
- Prioriza fuentes oficiales o claramente reutilizables.

## Forma de trabajar

- Haz cambios pequenos y directos.
- Conserva el repositorio simple salvo que haya una razon clara para complicarlo.
- Si anades o revisas APIs, intenta validar documentacion, acceso gratuito y estado real.
